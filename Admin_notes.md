#### Phase 0
- Goal
  - Phase 0 is where you turn "a good idea" into an "approved, fundable opportunity" with enough clarity on:
    - Who the target customer is.
    - What problem/need you're solving (supported by customer evidence).
    - What it will take (scope / cost / time triangle assumptions).
    - What could kill it (RISDA-driven risks + mitigations).
    - How we'll win commercially (early commercialization plan).
- Gate Inputs
  - Solution Development Charter
    - Owner: Technically Upstream Marketing, but project lead generally drives.
    - Content
      - Opportunity statement + target customer/persona.
      - Proposed solution concept + boundaries (in-scope / out-of-scope).
      - Success metrics (commercial + customer + quality/regulatory as applicable).
      - High-level plan + key milestones (ties to "Establish Key Project Milestones").
      - Top RISDA items (ties to "Create Project RISDA").
      - Resourcing + spend assumptions for next phases (so leadership knows what they're funding).
  - Customer Research Plan and Summary
    - Owner: Upstream Marketing.
    - Plan for gathering customer inputs (needs).
    - Summary of results.
  - Phase 0 Review Summary + Pre-Gate 0 Review Summary.
  - Commercialization Maturity Assessment (CMA).
  - RISDA.
- Outputs
  - MIP Gate 0 review.
  - Solution Development Charter approval ("go fund and go build" authorization).
- Governance
  - Phase Review led by Manager or Project Lead.
  - Pre-Gate Review led by Director Sponsor.
  - Gate Review led by Executive Sponsor.
  - Typical content for Pre-Gate + Gate:
    - Phase review recommendations.
    - Charter.
    - RISDA.
    - CMA (current + next-phase target).
- Minimum Deliverables
  - Customer + Needs evidence
    - `MIP-P0-001`: Establish Target Customer Personas.
    - `MIP-P0-002`: Create Customer Research Plan and Summary.
  - Commercial path
    - `MIP-P0-731`: Establish Commercialization Plan.
  - Project / program oversight backbone
    - Solution Development Charter activity (shown as Phase 0 charter work).
    - `MIP-P0-615`: Establish Key Project Milestones.
